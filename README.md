# 🚀 Adventure Works Data Analysis Project

> Adventure Works Sales & Customer Analysis using Power BI, Power Query, and DAX. Features dynamic KPIs and YoY growth tracking.

---

### 📝 Project Overview
This project provides a comprehensive **Business Intelligence (BI) solution** designed to analyze sales performance, customer behavior, and inventory management for the fictional retail giant, **Adventure Works**.

### 🎯 Project Objectives
* **Trend Analysis:** Analyzing monthly and annual sales cycles to identify growth patterns.
* **Customer Segmentation:** Profiling customers based on purchasing behavior.
* **Product Performance:** Evaluating KPIs across different product categories.
* **Executive Decision Support:** Developing interactive dashboards for senior management.

### 🛠 Tech Stack
* **Power BI Desktop:** Core tool for modeling and visualization.
* **Power Query:** Used for advanced ETL processes and data cleaning.
* **DAX:** Utilized for complex calculations and Time-Intelligence.
* **Data Modeling:** Implementation of a **Star Schema**.

### 🔍 Implementation Steps
1. **Data Extraction & Transformation:** Performed cleaning and adjusted data types in Power Query.
2. **Data Modeling:** Established relationships (1:N) and integrated a **Calendar Table**.
3. **DAX Calculations:** Developed measures like `Total Revenue`, `Total Profit`, and `YoY Growth`.
4. **Data Visualization:** Designed dashboards with slicers and dynamic trend charts.

### 📊 Key Dashboard Features
* **Executive Summary:** High-level overview of revenue and profit margins.
* **Product Detail:** Deep-dive analysis into top-performing products.
* **Customer Insights:** Detailed breakdown of demographics and geographic distribution.
